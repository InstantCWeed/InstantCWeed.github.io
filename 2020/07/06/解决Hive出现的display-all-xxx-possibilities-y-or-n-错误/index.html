解决Hive出现的display all xxx possibilities?(y or n)错误

2020-07-06 解决问题 0 评论

今天从notepad++中导入建表语句时，出现了“display all xxx possibilities?(y or n)”的错误。

原因如下：

解决Hive中出现 Display all 528 possibilities? (y or n)错误里面提到SQL建表语句中存在tab键导致，table在linux系统中有特殊含义。而在建表语句中习惯的使用tab键调整代码布局。上文给出的解决方法是将空格删去。

解决方法

我想去更改建表的语句总归不是长久之计，然后发现可以将notepad++中的tab键设为四个空格
里面提到了可以直接通过 设置-> 首选项 -> 语言 或者 设置 -> 首选项 ->制表符设置 将tab键转换成需要的空格数，解决问题。
