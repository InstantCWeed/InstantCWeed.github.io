idea———使用时遇到的问题汇总

2020-03-25 工具使用 0 评论

IDEA关闭\打开最近的项目

内容辅助键：
快速生成语句：1.快速生成main方法：pvsm,回车；
2.快速生成输出语句：sout,回车；
3.快速生成增强for循环语句：iter,回车
内容辅助键： ctrl+alt+space(内容提示、代码补全)
注释快捷键： 单行，选中代码，ctrl+/，再按一次取消；
多行，选中代码，ctrl+shift+/，再按一次取消；
整理格式：ctrl+alt+L
同时修改所有同名变量：shift+F6
删除整行：ctrl+Y
撤销删除整行：ctrl+shift+z
整体项目从上往下结构：项目->模块->包->类

IDEA的项目结构：

写出方法后，ctrl + alt + v 会自动补齐左边的内容
先写右边方法，然后自动补齐左边，返回一个变量用于接收

alt + Insert： 自动弹出类的构造方法、get和set方法、equal()和hashCode()等等非常常用的方法。
ctrl + o:自动弹出可以重写的方法列表

看方法源码:选中方法，按下Ctrl+B
以层级hierarchy的形式，看类的向上和向下继承关系:Ctrl+H
按照框图diagrams查看类间继承和接口调用的关系：Ctrl+alt+u

导入包或者加上异常处理：选中对象，按下Alt + enter

查找源码的具体功能函数：Alt+7：打开一个窗口，能够看到类的所有信息
View->Tool Windows->Structure(Alt+7);
然后在Structure的settings中，选择Move to right;
就能在窗口中找到想要的功能函数了。

对选中的多行代码进行 上、下、左、右 移动
首先都要多行选中
右移：Tab键
左移：shift + Tab
上移：shift + ctrl + 方向键上
下移：shift + ctrl + 方向键下

idea双击shift弹出的SearchEverywhere关闭

  1. control+shift+A；
  2. 输入registry；
  3. 找到ide.suppress.double.click.handler,点对勾。
