Hexo博客的站内跳转方法

2020-03-02 hexo使用 0 评论

通过查询之后，发现站内跳转有一个用法：

{% post_link 文章文件名(不要后缀) 文章标题(可选) %}

如 文章名为Hello-World.md，则写法为：

{% post_link Hello-World %}

或

{% post_link Hello-World 你好世界 %}

在哪里插入就直接写上面就行，不需要再加[]。
但是我没有找到类似像target=”_blank”这样可以弹出新窗口的方法..

其实，我之前还想到了另外 两个方法：
1) 利用MarkDown语法，就是用

[文字](地址)
这种写法，后面地址写跳转网站的绝对地址，感觉这种好用但是比较憨憨的…
2) 还有一种跟上面比较像，也是
[文字](地址)
这种写法，但是后面写那个另一个跳转的md文章的名称，但是需要放在同一个文件夹中，并且这个网站的路径可能会不一样，因为有些网站生成用的是时间做层级路径，需要改_config.yml中permalink标签，这个比较麻烦，还是最上面那个最适合。

此外，还有一种页内跳转的方法，类似点击目录，跳转到具体的标题处。
方法详见Markdown 如何实现页面内跳转
